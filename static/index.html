Curve to Contrast

From Kaplan-Meier Curves to Log-Rank Results.

This prototype website uploads a Kaplan-Meier image, uses a vision-capable LLM API to estimate event times and censor indicators, reconstructs pseudo-records, and computes a two-arm log-rank test in the browser. It also searches PubMed for treatment chains like A vs B and B vs C, then combines saved two-arm studies into an indirect A vs C comparison.

Workflow

  1. Upload a Kaplan-Meier figure and label the treatment arms.
  2. Send the image to an OpenAI Responses-compatible LLM endpoint.
  3. Review the reconstructed event table, pseudo-records, and warnings.
  4. Compute two-arm log-rank results and reconstructed survival curves.
  5. Optionally search PubMed and combine two saved studies through a common comparator.
